FAQ

What’s the purpose of MonetizeJS?

The purpose of MonetizeJS is to provide with user payment services applications that don’t have any web server or don’t want to implement user management.

What browser do we need to use MonetizeJS?

MonetizeJS platform requires a modern browser such as Chrome, Firefox or Internet Explorer 9 to run properly.

How much does it cost?

MonetizeJS applies a 5% fee on top of the Stripe processing fees.

How are users impacted by pricing option modifications?

When a developer modify a pricing option, users that already have paid are not impacted. It means that:

  • current once-off charge is evaluated to its original price.
  • current subscription will continue on its original plan.

Can we convert a subscription to a once-off charge?

No, once-off charges and subscriptions are not convertible. However, a developer can freely credit user’s total charges.
